pygmin.takestep.BlockMoves¶

class pygmin.takestep.BlockMoves¶

block based step taking

This class is a wrapper for takestep objects to group several takestep objects into one and perform takestep moves block wise, changine the takestep mechanism after a given amount of steps.

Takestep objects can be added with addBlock

step1 = RandomDisplacement()
step2 = RandomRotation()

step = BlockMoves()
step.addBlock(100, step1)
step.addBlock(100, step2)
