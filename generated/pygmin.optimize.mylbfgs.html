pygmin.optimize.mylbfgs¶

pygmin.optimize.mylbfgs(coords, getEnergyGradient, **kwargs)¶

A wrapper function for MYLBFGS.

This version uses the GMIN fortran code to update the Hessian approximation and generate a trial step. The actual step taking algorithm is the same as lbfgs_py.
