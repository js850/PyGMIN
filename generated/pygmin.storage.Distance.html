pygmin.storage.Distance¶

class pygmin.storage.Distance(dist, min1, min2)¶

object to store “mindist” distances between minima

Parameters :

dist: float :

distance between minima

min1: Minimum object :

first minimum

min2: Minimum object :

second minimum

See also

Database, Minimum, TransitionState

Notes

To avoid any double entries and be able to compare them, only use Database.setDistance() to create a Distance object.

programming note: As with TransitionState, functions in the database require that dist.minimum1._id < dist.minimum2._id. This will be handled automatically by the database, but we must remember not to screw it up.
