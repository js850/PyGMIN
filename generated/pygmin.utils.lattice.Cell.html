pygmin.utils.lattice.Cell¶

class pygmin.utils.lattice.Cell[source]¶

class to handle crystal cells

The internal representation of the lattice is in the gruber notation.

Methods

get_lower_triangular() calculate the lower triangular lattice matrix
get_upper_triangular() calculate the upper triangular lattice matrix
get_with_angles() get the cell parameters with box length / angles
set_lattice_matrix(lattice) set cell parameters from lattice matrix
set_with_angles(a, b, c, alpha, beta, gamma) set cell parameters from angle.
