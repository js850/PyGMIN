pygmin.landscape.DoubleEndedConnectPar¶

class pygmin.landscape.DoubleEndedConnectPar(*args, **kwargs)[source]¶

parallelized version of DoubleEndedConnect

Overload some of the routines from DoubleEndedConnect so they can be parallelized

Parameters :

ncores : int, optional

the number of cores to use in parallel runs

See also

DoubleEndedConnect
the class this inherits from
LocalConnectPar
parallel version of LocalConnect

Notes

This class inherits from DoubleEndedConnect, so can accepts all those parameters as well.

The routines that are done in parallel are:

  1. NEB : the potentials for each image are calculated in parallel
  2. findTransitionStates : each transition state candidate from the NEB run is refined in parallel.

Methods

connect([maxiter]) the main loop of the algorithm
getDist(min1, min2) get the distance between min1 and min2.
mergeMinima(min1, min2) merge two minimum objects
