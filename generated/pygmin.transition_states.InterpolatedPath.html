pygmin.transition_states.InterpolatedPath¶

class pygmin.transition_states.InterpolatedPath(initial, final, nimages, interpolator=<function interpolate_linear at 0x32dfde8>)¶

Wraps interpolation that it can be accessed like a list / array without storing the nodes

Parameters :

intitial: :

first point

final: :

second point

nimages: integer :

number of images

interpolator: callable :

interpolation algorithm

See also

InterpolatedPathDensity
use image density

Examples

>>> import numpy as np
>>>
>>> path = InterpolatedPath(np.array([0., 1.]), np.array([1.,0.]), 10)
>>> print len(path)  # prints 10
>>> print path[5]    # access 5th frame
>>> 
>>> for x in path: # it is also iteratable
>>>     print x
